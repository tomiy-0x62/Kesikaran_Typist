## keys.csv

shiftの扱いが少し特殊

-> 右シフトと左シフトでキーコードが違っているけど、その区別を外すめに421を自分で割り当てた。

区切り文字がスペースになっていることに注意

-> 文字コードを並列するためにカンマを使いたいから

一番左の列のkeyNumは押されたキーを光らせるためのもの

-> "を" "shift_0" みたいな組み合わせのについてはそれぞれ、[shift, 0]と出せばいい

-> 組み合わせのキーは全部100
